Инструкция для запуска демонстрационного приложения:
1. Открыть консоль.
2. В консоли перейти в директорию "demo/demo_app".
3. В консоли выполнить команду "python demo.py"
4. Возможно потребуется решить проблемы с безопасностью.
5. Открыть браузер и ввести в адресную строку "http://127.0.0.1/sentiment-demo".
6. Ввести в текстовое поле текст примера.
7. Нажать кнопку "Оценить".
